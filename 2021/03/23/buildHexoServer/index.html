Some Experience of Build Hexo Server

Yang
2021-03-23
hexo, experience

Install Hexo

If we want to use hexo command, we need to set the path of mode_modules with the code:

echo 'PATH="$PATH:./node_modules/.bin"' >> ~/.profile

Then we can use hexo init to create a hexo repository.

Deploy in Github

A _config.yml config basic infos about the sites. And some parameters need to take care:
Hexo creates an addtional fold when post_asset_folder is true.

Deploy section tells hexo the infos about deployer, the type, target url and the branch. 

deploy:   
  type: git  
  repo: https://github.com/yang-0209/yang-0209.github.io.git   
  branch: master

Intall the deployer before use it with the command
npm install hexo-deployer-git --save

Write Articles

There three important command:hexo clean, hexo g, hexo d, cleaning caches, generate article code and deploy contents to github.

Delete Articles

hexo list post shows all articles. Delete them under _post fold and deploy to server again.
